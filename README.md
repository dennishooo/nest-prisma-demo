## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Nest

```bash
# create resources
$ npx nest generate resource

# create filter
npx nest generate filter prisma-client-exception
```

## Prisma

```bash
# run migration
$ npx prisma migrate dev --name "init"

# run seed
$ npx prisma db seed

```
